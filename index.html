Full Enterprise IT Environment on Blade Server
Designed and deployed a complete production-equivalent enterprise IT environment from scratch on a physical blade server running VMware ESXi. The environment runs multiple virtualized systems simultaneously:

Windows Server 2025 - Active Directory Domain Services, DNS, DHCP, and Group Policy Objects enforced across all domain-joined machines. Windows 11 client VMs fully domain-joined with GPO policies applied. Ubuntu Server for Linux-based services. Network security managed by a pfSense firewall with a fully configured IPsec site-to-site VPN simulating a multi-branch business. Centralized storage handled by TrueNAS. A PBX phone system for internal voice communication. In-house email via HMailServer + Thunderbird, keeping all messages fully internal. IT operations managed through a GLPI ticketing system - a real help desk workflow from ticket open to resolution.
Rationale

This isn't a single-service lab - it's a fully integrated environment where every component connects to every other. It demonstrates mastery across virtualization, networking, security, storage, communications, and IT operations simultaneously. An employer can see that I can walk into a small-to-medium business and understand every layer of the stack on day one.

DNS & DHCP Server Configuration Lab
